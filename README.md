## About
Acti has been developped in order to be familiarized with new web technologies:

| For what?     | Used technology   |
|---------------|-------------------|
| JS library    | React             |
| Web framework | NextJS            |
| Backend API   | Vercel            |
| SaaS Database | Supabase          |
| Deployment    | Vercel            |
| CDN           | Supabase & Vercel |
| Code review   | SonarCloud        |
| Analytics     | Google Analytics  |
